# iterm2常用快捷键

＝＝＝
## 屏幕控制
- 全屏  command + enter

## 标签
- 垂直分屏：command + d
- 水平分屏：command + shift + d
- 切换屏幕：command + [ 或 command + ]

## 分屏
- 垂直分屏：command + d
- 水平分屏：command + shift + d
- 切换屏幕：command + [ 或 command + ]

## 编辑
- 到行首：ctrl + a
- 到行尾：ctrl + e
- 上一条命令：ctrl + p
- 搜索命令历史：ctrl + r
- 复制即选中：command + f
- 删除当前光标的字符：ctrl + d
- 删除光标之前的字符：ctrl + h
- 删除光标之前的单词：ctrl + w
- 清屏操作：ctrl + r

# 查看剪贴板历史
- command + shift + h
