# 日常使用命令总结

## sort
###  按列排序
-  -r: 以相反的顺序来排序
-  -u: 忽略相同行
-  -n: 以数字顺序来排序
-  -k: 是指定需要排序列数
-  -t: 指定栏位分隔符

```sh
查询线程cpu最高的情况
```
